Question 1:

Difference between map,filter,forEach,find is given below :-

map

1.map() calls a function once for each element in an array.
2.map() creates a new array from calling a function for every array element.
3.map() does not execute the function for empty elements.
4. map() does not change the original array.
5.map() method allows you to iterate over an array and modify its elements using a callback function

filter

1.The filter() method creates a shallow copy of a portion of a given array.
2.The filter() method does not execute the function for empty elements.
3.The filter() method does not change the original array.
4.If you check out the output of the above example, the value of y is an array of 1 element that satisfies the condition. This is because the filter() method iterates over all elements of the array and then returns a filtered array which satisfy the condition specified.

forEach

1.The forEach() method is an iterative method.
2.The forEach method passes a callback function for each element of an array together with the following parameters:
Current Value (required) - The value of the current array element
Index (optional) - The current element's index number
Array (optional) - The array object to which the current element belongs

find

1. the find() method searches for first element in the array that satisfies the condition specified.
2. filter() returns an array containing the element that satisfies the condition, but find() returns the element itself that satisfies the condition.
3. In filter(), whole array is iterated despite the fact that the element being searched for is present at the beginning. But in find(), as soon as the element that satisfies the condition is found, it gets returned.h1>
